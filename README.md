# Simon Says Game

This is a web-based Simon Says memory game built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat an increasing sequence of colored buttons.

## How It Works
- Press any key to start the game
- Watch the sequence of highlighted colors
- Click the buttons in the same order
- Each successful round increases the difficulty

## Features
- Interactive color buttons
- Keyboard-based game start
- Progressive difficulty levels
- Simple and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
